GetList(Senpai, String, Int32, Int32, String, String, String)

Creates an ApiRequest instance that returns a list of all anime or manga a user has listed in their ucp.

Api permissions required:

  • UCP - Level 0
Declaration
public static ApiRequest<ListDataModel[]> GetList(Senpai senpai, string kat = "anime", int page = 0, int limit = 100, string search = "", string searchStart = "", string sort = "stateNameAsc")
Parameters
Type Name Description
Senpai senpai

The user of which the anime or manga are returned. Must be logged in.
System.String kat

Optional. The category that should be loaded. Possible values: "anime", "manga". Default: "anime"
System.Int32 page

Optional. The index of the page that should be loaded. Default: 0
System.Int32 limit

Optional. The number of entries that should be returned per page. Default: 100
System.String search

Optional. The string that all returned entries should contain. Default: ""
System.String searchStart

Optional. The string that all returned entries should start with. Default: ""
System.String sort

Optional. The order in which the returned entries should be returned. Possible values:

  • "nameASC": Order by entry name ascending.
  • "nameDESC": Order by entry name descending.
  • "stateNameASC": Order first by status of the entry and then by entry name ascending.
  • "stateNameDESC": Order first by status of the entry and then by entry name descending.
  • "changeDateASC": Order by last changed ascending.
  • "changeDateDESC": Order by last changed descending.
  • "stateChangeDateASC": Order first by status of the entry and then by last changed ascending.
  • "stateChangeDateDESC": Order first by status of the entry and then by last changed descending.

Default: "stateNameASC"
Returns
